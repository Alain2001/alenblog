In een stylesheet staat het volgende beschreven:

  • De fontgrootte van hoofding h1 staat op 2ex
  • De fontgrootte van hoofding h2 staat op 2em

Welk van beide hoofdingen wordt volgens de CSS standaard het grootst weergegeven en waarom?

Volgens de CSS standaard zal de fontgrootte h1 met 2ex het grootst weergegeven worden, omdat die de element h1 bevat en die wordt het grootst weergegeven als headings
